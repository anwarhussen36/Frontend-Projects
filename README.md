✅ To-Do List App
Another project included in this portfolio is a To-Do List App — a simple yet functional task management app built using React.js.

🧰 Features
✅ Add new tasks with ease

📝 Edit existing tasks

❌ Delete tasks

📌 Mark tasks as completed/incomplete

📱 Fully responsive design


🧑‍💻 Tech Used
React.js (Functional Components + Hooks)

CSS3 for styling

LocalStorage for saving tasks

JSX for UI structure



🧪 How to Use
Type a task in the input field and press “Add” or hit Enter.

Click the checkbox to mark a task as complete.

Use the edit icon to modify, or trash icon to delete a task.

🚀 How to Run Locally
# Clone the repo
git clone https://github.com/anwarhussen36/Frontend-Projects.git

# Navigate into the project directory
cd Frontend-Projects.
and select the ToDo-List app project and open it.

# Install dependencies
npm install

# Start the development server
npm start
